# MySQL Database Operations using Node.js

## Description
This project demonstrates basic MySQL database operations using Node.js and the mysql2 package.

## Operations Performed
- Connect to MySQL Server
- Create Database
- Use Database
- Create Table
- Insert Records
- Select All Records
- Select Record by ID
- Delete Record
- Add New Column
- Drop Column
- Update Record
- Drop Primary Key (optional)
- Drop Table

## Technologies Used
- Node.js
- MySQL
- mysql2 package

## How to Run

1. Install Node.js.
2. Install mysql2 package:

```
npm install mysql2
```

3. Start MySQL Server.

4. Run the program:

```
node mysql_operations.js
```
